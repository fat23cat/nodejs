# Users API

## Install dependencies

run `npm i`

## Data

Install Postgres server and pgAdmin 4
Run `npm run migrate` and `npm run seed` to create tables and fill them
Run `npm run seed:undo` and `npm run migrate:undo` to remove data and tables

## Run server app

`npm run server`
